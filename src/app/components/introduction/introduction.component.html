Getting Started

In order to start using mayajs you must install the latest stable version of Nodejs on your local machine. Having a prior knowledge in Typescript or Angular will help you alot. Like angular, mayajs uses typescript and much of the syntax implemented are heavily inspired from it.

Installation

The first thing we need to do is to install mayajs CLI globally in your local machine for project scaffolding.

npm i @mayajs/cli -g

After installing the CLI we need to check if mayajs is recognized by the machine. This command will show the current version of mayajs installed in your machine.

maya -v

This command will show all the available commands that can be use for your mayajs app.

maya

Now mayajs is properly installed to your machine. It is time to create a basic mayajs project. For this purpose we will use mayajs CLI commands to generate the files necessary for you to run your first mayajs application.

maya new "name-of-your-app"
