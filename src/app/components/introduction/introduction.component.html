What is Maya JS?

is a NodeJS framework for creating RESTful API application that follows the same syntax architecture of Angular.

Installation

The first thing we need to do is to install globally in your local machine for project scaffolding.

npm i @mayajs/cli -g

After installing the CLI we need to check if is recognized by the machine. This command will show the current version of mayajs installed in your machine.

maya -v
0.6.3

Now that you have confirmed that MayaJS CLI is installed, you are now ready to start your REST API Project.
