Phase 1: Project Initialization by Scaffolding

Making our REST API boiler plate is very easy thanks to . All you have to do is call this command on your terminal in your project directory 

          {{directory.outer}}
          maya

This is your menu. You can use your arrow key up and down to navigate and then press the Enter key to select. In this instance, you will create a project. Name the project "games-api" (quotation excluded) then select mongo template. It will install and scaffold the boilerplate and all its dependencies.
