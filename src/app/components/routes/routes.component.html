Components

Theses are the building block for creating a MayaJS application. You can compose these components to create routes.

Controllers

A controller acts a route that has its own sub-routes which handles all the incoming request from the client. You can define a sub-routes using method decorators that describe what kind of request it will accepts. You can also add middlewares and validations inside of these method decorators. See method decorators for more information on middlewares and validation.

To generate a controller you can run this command in your terminal.

maya generate controllers "controller-name" || maya g c "controller-name"

This will create a controller file based on the controller-name provided. Below shows a codeblock that generated inside the controller.

Routes

This module handles all the routes for our server. You can define a path for your controllers. You can also add middlewares for your route to use. These middlewares will be executed first before your controller.

maya generate route "route-name" || maya g r "route-name"

This will creates a controller, model and service. This will also create a folder from the given route-name and put all the created files inside. Each created files will have the same naming convention based on the route-name.
